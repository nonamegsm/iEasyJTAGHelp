Introduse

The EasyJtag it's hardware and software complex which intended for the automated restoration of firmware of mobile phones, smartphones, tablets and other electronic devices, hereinafter referred to as devices using the JTAG interface and/or eMMC memory chips.

Hardware 

The hardware is represented by two solutions. It's EasyJtag box and EasyJtagPlus box.

EasyJTAG box is first generation box which support:

  • JTAG connection (with native jtag 20pin cable)
  • ISP (eMMC at PCB) connection (with additional third party adapter)
  • eMMC e-Socket connection (1-bit mode with additional third party adapter)
  • SPI connection (with additional external chip power (1,8 or 3,3v) adapter)

EasyJTAG Plus box is second generation box which support:

  • JTAG connection (with native Z3x adapter)
  • eMMC e-Socket connection (with native Z3x e-Socket adapter or additional third party e-Socket adapter)
  • ISP (eMMC at PCB) 1-bit mode connection (with native Z3x adapter)
  • SPI connection (with additional socket or soldering adapter)
  • NAND connection (with additional NAND Kit adapter)
  • M-PHY (UFS Memory) connection (with additional special UFS Memory socket adapter)
