eMMC software structure

eMMC Operations

 

eMMC card have next internal ROMs structure:

ROM1 (User Area Partition) - Main Partition
ROM2 (Boot 1 Partition) - Partition for external boot (preloader)
ROM3 (Boot 2 Partition) - Partition for external boot (preloader)
RPMB partition (Protected Memory Block) - - Partition for store secure data

ROM1 partiton can have additional GP1-GP4 (General Purpose) Partition (reduces ROM1 size)
