USB Page

Software GUI

ADB - allows you to work with the device's memory through the ADB interface: Android Debugging Bridge, which is enabled in the device in the Developer Setting > USB Debuging.

Detect ADB - find/connect to the ADB interface of the device;

Detect Part. - identify device sections;

Dump Part. - read and save the selected section;

Read extCSD - read the configuration of the registers extCSD of the microchips memory device (eMMC) to the file;

Limit Dump Size To - limits the size of the selected section;

Partition List - illustrates the Symlink, Name, Size and Offset of partitions.

 

 

QC EDL tab is working with Qualcomm processors.

Qualcomm processors have a service mode that can be accessed by shortening a test pins.

Than the processor goes into boot mode and allows user to record the bootloader and trough it to perform the settings.

The sequence of operations :

1) Port - Allows to select a specific device (phone) that is connected via USB.

The button on the right allows to update the list of connected devices and automatically select the first QDLoader 9008 (the phone is in service mode).

2) Identify - allows to get the device status, ID and hash.

3) Loader - allows to select one of the compatible (by ID and hash) loaders from EasyJtag database, or use the Custom button to select the user bootloader in the box below.

4) The next time click the Identify button, a corresponding bootloader will be loaded into the device, allowing user to work with memory.

Reset Device - in some loaders and phones it allows you to switch from service mode to working mode . It doesn't always work, it's the features of a particular phone and Loader;

Read Info - reloading partitions;

Write Target – write to device.   The nuance of WriteMode:

Write Binary Raw to 0 sector - offset linear recording;

Write XML Write by vendor - parses the main firmware file and writes the corresponding files to the specified offsets;

Write by partition name - reads GPT from a device, locates a file with the same name in the selected folder and writes to it;

GPT Backup Writing – backup the GPT;

Read RAW – linear reading;

Read XML - reads files by analogy as they are placed in the firmware;

Unlock User - attempt to reset the user's password;

Stop Operation – stops the device operation .
