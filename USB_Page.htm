USB Page

Software GUI

ADB - allows you to work with the device's memory through the ADB interface: Android Debugging Bridge, which is enabled in the device in the Developer Setting > USB Debuging

Detect ADB - find/connect to the ADB interface of the device;

Detect Part. - identify device sections;

Dump Part. - read and save the selected section;

Read extCSD - read the configuration of the registers extCSD of the microchips memory device (eMMC) to the file;

Limit Dump Size To - limits the size of the selected section;

Partition List - illustrates the Symlink, Name, Size and Offset of partitions.
