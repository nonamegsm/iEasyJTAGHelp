Supported Operations and Modes

Overview

 

EasyJTAG Plus box support next operation modes:

JTAG mode connection
NAND mode connection
eMMC ISP mode connection
eMMC socket mode connection
UFS socket mode connection
SPI interface mode connection
USB Qualcomm QDLoader 9008 port connection
USB LG LAF mode port connection
USB ADB (Android Debug Bridge) mode connection
