Settings Page

Software GUI

 

 

 

"Firmware/Fullflash converter"  - on development stage;

"eMMC Tool Options"  - eMMC chip settings;

"eMMC Bad Block Retry Count"  - setting the number of read retries of badly read or unreadable blocks in eMMC;

"eMMC Bad Block Retry Delay"  - setting the timeout when reading bad or unreadable blocks in eMMC;

"eMMC Erase Block Size"  - eMMC block size when erased; 

"Full eMMC Reinit after Bad Block Detected"  - whether to do a full reset (initialization of connection) of eMMC upon detection of badly readable or unreadable blocks in eMMC;

"Disable High Speed eMMC timing"  - ban on the operation of the chip in the "High Speed Timing" mode (one of the extCSD eMMC registers is responsible for working in this mode;
